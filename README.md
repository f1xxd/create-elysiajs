# create-elysiajs

## Scaffolding your Elysia project with the environment with easy!

### With bun

```bash
bun create elysiajs
```

## Supported environment

-   Linters
-   -   [ESLint](https://eslint.org/)
-   -   [Biome](https://biomejs.dev/)
-   And more soon...

> Support for other package managers will appear later
